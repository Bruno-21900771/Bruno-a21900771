Sobre esta Aplicação

1. Arquitetura MVT do Django

O Model representa os dados e a lógica de acesso à base de dados (ex: a classe Tecnologia em models.py). A View recebe o pedido HTTP do utilizador, comunica com o Model para obter ou guardar dados, e escolhe o Template certo para responder. O Template é o ficheiro HTML que define como a informação é apresentada ao utilizador no browser.
